System Calls

int open(const char *pathname, int flags)
int open(const char *pathname, int flags, mode_t mode)

pathname - Paths to the file

flags - These affect how the file is opened

Bitwise OR | can be used to specify more than 1 flag

O_RDONLY - Read only
O_WRONLY - Write only
O_RDWR - Read and write access

O_TRUNC - When writing to an existing file, get rid of the previous data
O_APPEND - Star writing to the end of the file
O_CREAT - If the file doesnt exist create it 

mode - If you specify the mode here, then any new file that is created will have this mode
