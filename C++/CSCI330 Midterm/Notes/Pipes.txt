Pipes

Pipes which you create with the pipe system call are basically temporary files that exist in memory.

int pipe(int pipefd[2]);

Success: Returns file descriptor 
Fail: Returns -1 

pipefd[0] : File descriptor to read from the pipe
pipefd[1] : File descriptor to write to the other side of the pipe

Creates a channel to transport data 

2 processes communicate via pipe

Example:

ls | wc 

parent runs ls 

childs runs wc
