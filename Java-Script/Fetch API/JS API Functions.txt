JS Api Functions

FETCH API:
	The modern way to make HTTP requests in JavaScript, which is built into most modern browsers

	fetch(url)
	.then(response => response.json())
	.then(data => console.log(data))
	.catch(error => console.error("Fetch error", error));

Axios:
	A promose-based HTTP client for the browser and Node.js. It simplifies making HTTP requests 	and handlin responses

	axios.get(url);
	.then(response => console.log(response.data));
	.catch(error => console.error("Axios error", error));

Async/Await:
	async function fetchData() {
	  try {
            const response = await fetch(url);
            const data = await response.json();
            console.log(data)
           } catch (error)
              console.error("Async/Await error:", error);
           }
	 }

Promises:
	function fetchData() {
	  return new Promise((resolve, reject) => {
		fetch(url)
		.then(response => resolve(response.json())
		.catch(error => reject(error));
         });
	}
